Detailed Description

IMU cluster log.

The purpose of this log is to be used in production.

The calibration procedure has to be the same for all IMUs. It must not know the specifics of the IMUs.

An IMU does not necessarily use all axes of its physical sensors, which would require knowledge of the specifics of the IMU.

As a result, values of the sensors are split per axis instead of (x, y, z) tuples. In other words, xAccels[i], yAccels[i] and zAccels[i] can come from different sensor.

This is a public structure.
